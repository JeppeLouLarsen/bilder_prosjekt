RGB CSS

RGB er systemet man bruker i css for å lage forskjellige farger. Måten man oppnår disse nye fargene er ved å kombinere fargene rød, grønn og blå sammen, som man ser på bildet til siden. Fargen hvit har fargekoden 255, 255, 255. Og sort har derfor fargekoden 0, 0, 0. Og for alle andre farger så er det bare å blande forskjellig mengder av de tre fargene helt til man får den fargen man leter etter. Hvis man øver nok vil man kanskje til slutt å lage akuratt den fargen man er ute etter.
